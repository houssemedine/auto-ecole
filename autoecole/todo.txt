Checklist Passage en Production Django
DJANGO_SETTINGS_MODULE=config.settings.development python manage.py runserver 0.0.0.0:8000
DJANGO_SETTINGS_MODULE=config.settings.production python manage.py runserver

1) Env & settings
   - SECRET_KEY fort dans .env
   - ALLOWED_HOSTS réels
   - DEBUG=False (déjà forcé)
   - CORS_ALLOWED_ORIGINS / CSRF_TRUSTED_ORIGINS en https

2) Dépendances
   - MAJ vers Django/DRF/SimpleJWT compatibles (fait)
   - Geler requirements.txt

3) Static/Media
   - python manage.py collectstatic --noinput
   - Nginx sert /static/ et /media/ (ou WhiteNoise si pas d’Nginx)

4) Base de données
   - (Si Postgres plus tard) DATABASE_URL + user non-superuser + sslmode=require
   - python manage.py migrate --noinput

5) Serveur d’app
   - Gunicorn (WSGI) ou Uvicorn (ASGI) sous systemd/Docker

6) Reverse proxy & HTTPS
   - Nginx (ou Traefik/Caddy) 80→443, certbot OK
   - Quand HTTPS OK → SECURE_SSL_REDIRECT=True + SECURE_HSTS_SECONDS=31536000

7) Sécurité
   - Admin URL non triviale (ADMIN_URL)
   - Comptes staff 2FA (si possible)
   - python manage.py check --deploy propre

8) Observabilité
   - Logs visibles (journalctl / docker logs)
   - Sentry (optionnel) via SENTRY_DSN

9) Sauvegardes
   - Backups DB + media
   - Plan de rollback (release N-1)

10) Tests finaux
   - Smoke test endpoints clés
   - CORS depuis le front OK
   - Uploads/téléchargements OK
